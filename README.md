# my-first-project
# AI智能伴侣 使用说明书
## 一、功能概述
本程序是基于 Streamlit 开发的 AI 聊天应用，核心功能为模拟「青梅竹马」风格的 AI 智能伴侣对话，支持自定义 AI 昵称、性格，同时具备会话的新建、保存、加载、删除等管理功能，对话采用流式输出，更贴近真实聊天体验。

## 二、运行环境与前置条件
### 1. 环境要求
- Python 3.8 及以上版本
- 操作系统：Windows/macOS/Linux 均可

### 2. 依赖安装
在终端/命令行执行以下命令安装所需依赖：
```bash
pip install streamlit openai python-dotenv
```

### 3. 密钥配置
本程序调用 DeepSeek 大模型 API，需配置 API 密钥：
- 方式 1（推荐）：在系统环境变量中添加 `DEEPSEEK_API_KEY`，值为你的 DeepSeek API 密钥；
- 方式 2：在代码中直接替换 `os.environ.get('DEEPSEEK_API_KEY')` 为你的密钥（不推荐，存在密钥泄露风险）。

## 三、使用步骤
### 1. 启动程序
将代码保存为 `.py` 文件（如 `ai_partner.py`），在终端/命令行进入代码所在目录，执行：
```bash
streamlit run ai_partner.py
```
程序会自动打开浏览器，显示 AI 智能伴侣页面（默认地址：`http://localhost:8501`）。

### 2. 基础操作
#### （1）对话交互
- 在页面底部的输入框中输入想和 AI 伴侣说的话（如「今天天气真好～」），按回车发送；
- AI 会按照「青梅竹马」角色设定和你输入的性格特征，以流式方式回复消息，对话记录会实时显示在页面上。

#### （2）自定义 AI 伴侣信息
在左侧侧边栏「AI 伴侣信息」区域：
- 「昵称」输入框：修改 AI 伴侣的名字（默认：冰冰）；
- 「性格」输入框：自定义 AI 伴侣的性格特征（默认：纯真温柔的 17 岁江南女孩），修改后即时生效。

#### （3）会话管理
在左侧侧边栏「会话历史」区域：
- 新建会话：点击「新建会话」按钮，当前会话会自动保存，同时创建全新的空白会话；
- 加载历史会话：点击会话名称按钮（如 `2026-03-11_15-30-20`），即可加载该会话的所有聊天记录；
- 删除会话：点击会话名称右侧的「❌」按钮，即可删除该历史会话（删除后无法恢复）。

## 四、注意事项
### 1. 数据存储相关
- 会话数据以 JSON 文件形式保存在代码所在目录的 `session` 文件夹中，删除该文件夹会丢失所有历史会话；
- 每次发送消息、新建会话、修改 AI 信息后，程序会自动保存当前会话数据，无需手动操作。

### 2. 配置与兼容性
- 代码中 `st.logo(r"C:\Users\苏铭\Pictures\辉夜姬.png")` 为本地图片路径，需替换为你自己的图片路径（或删除该行），否则会因图片不存在报错；
- 若运行时提示「找不到 DeepSeek API 密钥」，需确认 `DEEPSEEK_API_KEY` 环境变量已正确配置，或 API 密钥未过期；
- 不同 Streamlit 版本可能存在界面显示差异，建议使用 1.30.0 及以上版本。

### 3. 功能限制与风险
- 仅支持 DeepSeek 大模型，若需切换其他模型（如 OpenAI GPT），需修改 `client` 初始化的 `base_url` 和 `model` 参数；
- 流式输出仅支持 DeepSeek 模型的流式响应格式，更换模型需适配响应解析逻辑；
- 请勿将包含 API 密钥的代码上传至公开仓库（如 GitHub），避免密钥泄露导致财产损失。

### 4. 异常处理
- 若页面无响应或报错，可尝试关闭浏览器后重新执行 `streamlit run ai_partner.py`；
- 加载/删除会话失败时，页面会显示错误提示，可检查 `session` 文件夹权限（如是否为只读）。

### 5. 其他
- 程序默认会话名称为「当前时间戳」（格式：年-月-日_时-分-秒），暂不支持自定义会话名称；
- AI 回复严格遵循代码中的系统提示词规则（如「回复简短、匹配用户语言、使用 emoji 等」），修改规则需调整 `system_prompt` 变量。

## 总结
1. 核心使用流程：安装依赖 → 配置 API 密钥 → 启动程序 → 自定义 AI 信息 → 对话交互 → 管理会话；
2. 关键注意点：替换本地图片路径、妥善保管 API 密钥、注意会话数据的本地存储位置；
3. 功能扩展建议：如需自定义会话名称、切换大模型，需修改代码中对应的变量/参数。
